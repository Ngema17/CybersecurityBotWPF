## Project Overview
This is a WPF-based Cybersecurity Awareness Chatbot developed for PROG6221 POE Part 2. The chatbot helps users learn about cybersecurity through an interactive graphical interface.

## Features
### Core Features
-  Modern WPF GUI with professional dark theme
-  ASCII art integrated into GUI
-  Voice greeting (WAV audio playback)
-  User name input and personalized greetings

### Part 2 Enhancements
-  **Keyword Recognition:** Detects passwords, phishing, scams, privacy, and browsing topics
-  **Random Responses:** Multiple response variations for each topic
-  **Conversation Flow:** Handles "tell me more", "another tip", "explain more"
-  **Memory Feature:** Remembers user's favorite cybersecurity topics
-  **Sentiment Detection:** Recognizes worried, frustrated, and curious sentiments
-  **Error Handling:** Graceful handling of unknown inputs
-  **Typing Animation:** Realistic typing effect for bot responses
